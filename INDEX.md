# Project Index

**Ring Attractor Recovery Under Partial Observation**

This document maps every file, dataset, checkpoint, and figure to its purpose.

---

## Source Code (`src/`)

### Simulator (`src/ring_attractor/`)
| File | Purpose |
|---|---|
| `defaults.py` | Single source of truth for all simulator + spike pipeline parameters |
| `network.py` | `RingAttractor` class — cosine kernel, tanh(4*ReLU) nonlinearity, Euler integration |
| `spiking.py` | `SpikeProcessor` — Poisson spikes, temporal binning, causal smoothing |
| `plotting.py` | Polar snapshots, heatmaps, trajectory plots |
| `__init__.py` | Package exports |

### Training (`src/train/`)
| File | Purpose |
|---|---|
| `models.py` | `VanillaRateRNN`, `LowRankRateRNN`, `create_model()` |
| `dataset.py` | `RingAttractorDataset` — per-trial K randomization, observed neuron selection |
| `training.py` | `train()` + `TrainingConfig` — full training loop with noise annealing, convergence weighting |
| `evaluation.py` | All metrics: MSE, ring score (uniformity/circularity/spread), generalization drift, eigenvalues |
| `__init__.py` | Package exports |

### Standalone scripts (`src/`)
| File | Purpose |
|---|---|
| `generate_dataset.py` | Generate training data: 72 angles x 2 groups = 144 trials |
| `validate_simulator.py` | 5-test validation suite for the simulator |
| `visualize_ring_attractor.py` | Generate 6 diagnostic figures for the simulator → `figs/01_simulator/` |
| `demo_ring_attractor.py` | Jupytext demo notebook for the simulator |
